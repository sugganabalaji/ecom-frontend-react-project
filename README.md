# 🛒 ecom-products-react-ui is a React.js E-Commerce Project maintains Products Details

A modern e-commerce web application built with **React.js**. This project demonstrates a scalable frontend setup with support for product listings, cart management, and responsive UI.

---

## 🚀 Getting Started

Follow these steps to set up and run the project locally:

### 1. Check Node.js Version
Make sure you have Node.js installed.  
Run the following command:
```bash
  node --version
```
📖 Reference [Download Node.js](https://nodejs.org/en/download)

---

### 2. Check npm Version
Verify that npm (Node Package Manager) is installed:

```bash
  npm --version
```
📖 Reference: [About npm](https://docs.npmjs.com/about-npm)

---

### 3. Install Dependencies
Navigate to the project folder and install required packages:
```bash
  npm i
```
📖 Reference: [npm install docs](https://docs.npmjs.com/)
